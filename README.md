# 🧠 World's First Hybrid Calculator 🔥  
Smart Scientific + Voice + Task-Based Calculator

---

## 🌟 Overview  

This is not just a calculator… it’s a **Hybrid Smart Calculator** that combines:

- 🧮 Scientific Calculations  
- 🎤 Voice Input & Output  
- 📊 Task-based History Management  
- 🌗 Light/Dark Mode UI  
- 🔍 Smart Search System  

Built using **HTML, CSS, JavaScript + Math.js + Web Speech API**, this project transforms a simple calculator into a **powerful productivity tool**.

---

## 🚀 Key Features  

### 🧮 Advanced Scientific Calculator  
- Trigonometric functions (sin, cos, tan)  
- Logarithmic calculations  
- Power & root operations  
- Real-time evaluation  

### 🎤 Voice AI Integration  
- Voice input (English + Hindi)  
- Converts speech → mathematical expression  
- Speaks result using Text-to-Speech  

### 📂 Task-Based History System  
- Create multiple tasks (e.g., Budget, Study, Project)  
- Store calculations separately for each task  
- Day-wise structured history  

### 🔍 Smart Search  
- Search history by:
  - 📅 Date  
  - 📌 Task name  

### 🌗 Modern UI/UX  
- Dark & Light mode  
- Clean and responsive design  
- Smooth animations  

### 📈 Intelligent Features  
- Real-time calculation preview  
- Auto error handling  
- Smart percentage logic  
- Keyboard support  

---

## 🧠 Tech Stack  

- **Frontend:** HTML5, CSS3, JavaScript  
- **Library:** Math.js  
- **APIs:**  
  - Web Speech API (Voice Input/Output)  
- **Storage:** LocalStorage
